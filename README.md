# Resistance: An Artists Manual
This repo contains the source files for the Resistance: An Artists Manual Project by Poppy DeltaDawn and the Unstable Design Lab as well as a collection of downloadable 3D print models for weaving accessories. You can find the online resource at [resistance.unstable.design](https://resistance.unstable.design), which includes essays and how-tos for warping and weaving on TC2 looms.
